• É análogo ao Set,  porém só podemos adicionar valores que sejam Objetos.

• O objeto existe no Set enquanto é alcançavel de algum outro lugar do código

• O WeakSet suporta os métodos add,has e delete. Não aceita size() , keys() nem laços/iterações

• Serve como um "armazenamento" extra de dados, não para dados arbitrários, mas para fatos sim/não
