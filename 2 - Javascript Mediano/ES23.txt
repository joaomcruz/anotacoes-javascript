
•  findLast()

 => Retorna o último item dentro de um array




• findLastIndex()

 =>  Retorna o index do último item dentro de um array



• toReversed()

 => Inverte o array, sem alterar o array original


• toSorted()

 => Ordena em ordem alfabética sem alterar o array original


• toSpliced()

 => O primeiro argumento serve para selecionarmos em qual index começar a "remover" item do array.
    O segundo argumento é para selecionarmos a quantidadade de itens que serão removidos.
 
 => Remove sem alterar o array original.



 • with()

 => Primeiro parâmetro selecionamos o index de qual item queremos alterar.
 => O segundo argumento é de fato com o que queremos alterar (novo valor)
